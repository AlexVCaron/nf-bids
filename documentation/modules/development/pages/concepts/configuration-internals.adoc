= Configuration Internals
:navtitle: Configuration Internals

[.lead]
This page focuses on the implementation internals behind configuration parsing and validation.

== Main Classes

* `BidsConfigLoader` parses YAML with SnakeYAML and enforces reserved-key rules.
* `BidsConfigValidator` performs structural validation and emits errors/warnings.
* `BidsConfigAnalyzer` computes handler activation flags and `loop_over` entities.

== Internal Flow

. `BidsConfigLoader.load(configPath)` parses the YAML map.
. `BidsConfigValidator.validate(config)` verifies set-type and field rules.
. `BidsConfigAnalyzer.analyzeConfiguration(config)` determines active set handlers.
. `BidsHandler` consumes this analysis to route to handler implementations.
