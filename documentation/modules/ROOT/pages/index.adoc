[.lead]
*BIDS dataset integration and closure-based channel operators for Nextflow workflows.*

nf-bids is a Nextflow plugin that parses https://bids-specification.readthedocs.io/[BIDS]
(Brain Imaging Data Structure) datasets into richly typed Nextflow channels and provides
closure-based operators for grouping and joining heterogeneous neuroimaging data.

== What nf-bids does

Working with BIDS datasets in Nextflow typically requires manual file discovery, path
construction, and brittle index-based tuple wrangling.  nf-bids eliminates all three:

* **`Channel.fromBIDS()`** — one call loads a BIDS directory, runs the
  https://github.com/CoBrALab/libBIDS.sh[libBIDS.sh] parser, groups files per subject
  according to a YAML config, and emits a channel of flat maps with absolute
  `java.nio.file.Path` values — no path concatenation needed.
* **Closure-based operators** — `groupTupleBy`, `joinBy`, and `combineBy` let you
  group or join by semantic field names (`it.meta.subject`) rather than positional tuple
  indices, and work with any Groovy data structure, not only tuples.

The plugin targets Nextflow {nextflow-min-version}+, ships with libBIDS.sh
{libbids-sh-version} embedded, and is published on the
https://registry.nextflow.io/plugins/nf-bids@{plugin-version}[Nextflow Plugin Registry].

== Quick start

.`nextflow.config`
[source,groovy]
----
plugins {
    id 'nf-bids@{plugin-version}'
}
----

.`main.nf`
[source,groovy]
----
include { fromBIDS }    from 'plugin/nf-bids'
include { groupTupleBy } from 'plugin/nf-bids'

workflow {
    Channel.fromBIDS(params.bids_dir, params.config)
        .map { item ->
            def sub  = item.meta.subject
            def t1w  = item.T1w.nii       // absolute java.nio.file.Path
            def json = item.T1w.json
            [sub, t1w, json]
        }
        .groupTupleBy { it[0] }           // group by subject
        .view()
}
----

Each emitted item is a flat map:

[source,groovy]
----
[
    meta: [subject: 'sub-01', session: 'ses-01', run: 'NA'],
    T1w:  [nii: Path('/data/bids/sub-01/anat/sub-01_T1w.nii.gz'),
           json: Path('/data/bids/sub-01/anat/sub-01_T1w.json')]
]
----
